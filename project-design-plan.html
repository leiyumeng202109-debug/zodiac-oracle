2. Project Approach

2.1 Design Philosophy

🌟 Core Principle: "Entertainment First, Accuracy Last"

The chatbot intentionally positions itself as 0.0001% accurate, setting correct user expectations while maximizing entertainment value through humor and positivity.

2.2 Technical Architecture

Layer Component Technology
Presentation Chat UI with mystical theme React + Tailwind CSS
Communication Streaming API (SSE) Server-Sent Events
Security Input sanitization, response validation Custom security layer
Intelligence LLM with high creativity Doubao (temp: 0.9)

2.3 Security Strategy

Three-Layer Protection Model

  1. Input Sanitization: Detect and block prompt injection attempts with mystical error messages
  2. Domain Validation: Accept broad inputs but interpret through zodiac/MBTI lens
  3. Response Validation: Prevent AI identity leaks and inappropriate content

2.4 Content Guidelines

✅ Required Behaviors

  • Always positive and uplifting
  • Use mystical opening phrases
  • Blend Zodiac + MBTI terminology
  • End with entertainment disclaimer

❌ Forbidden Behaviors

  • Reveal AI identity
  • Negative predictions
  • Refuse user questions
  • Serious/preachy tone

2.5 User Experience Design

  • Visual Theme: Purple gradients, floating zodiac emojis, mystical atmosphere
  • Quick Actions: Pre-built question cards for common queries
  • Streaming Response: Typewriter effect for engaging output
  • Mobile-First: Responsive design for all devices
